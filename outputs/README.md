# Outputs

This folder is filled when you run `phase2_notebook.ipynb` or `python run_phase2.py`.

Expected generated files:

- `phase2_results.csv` - final numeric table
- `phase2_report.md` - automatically generated results interpretation
- `plots/` - training curves, confusion matrices, per-class accuracy plots, filter visualizations
- `histories/` - JSON training histories
- `models/` - optional model checkpoints, ignored by Git because they can be large
